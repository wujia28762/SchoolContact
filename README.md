## 项目介绍
### 角色
> 教师、家长、学生。
### 功能
> 以融云IM功能为核心，推送内容包括：作业消息、通知消息、聊天内容等。
> 班级圈类似于朋友圈，通过数据库关系屏蔽非本班级范围内的班级圈信息。
> 教师角色拥有发送通知、作业、班级圈消息的权限。和其他非核心功能。
> 家长和学生拥有接收联系人列表中消息的权限，对于本班级的同学可以发起会话的权限。
> 使用友盟SDK收集错误，使用信鸽推送进行消息的推送。
